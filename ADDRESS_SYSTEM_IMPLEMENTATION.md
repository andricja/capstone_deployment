## 🎯 How It Works

### User Flow

1. **User clicks "Request Rental"**
   - Modal opens
   - Municipalities load automatically

2. **User fills farm size and contact info**
   - Auto-calculates rental days and cost

3. **User selects address (4 steps)**
   ```
   Step 1: Type sitio/street → "Purok 1"
   Step 2: Select municipality → "Calapan City"
   Step 3: Select barangay → "Poblacion" (loads after municipality)
   Step 4: Province → "Oriental Mindoro" (fixed)
   ```

4. **User enters coordinates**
   - Required for transportation fee calculation
   - Can use GPS or map picker

5. **User submits**
   - Frontend sends structured fields
   - Backend generates: "Purok 1, Barangay Poblacion, Calapan City, Oriental Mindoro"
   - Calculates transportation fee based on distance

### Data Flow

```
Frontend                    Backend
--------                    -------
sitio_street: "Purok 1"  →  sitio_street: "Purok 1"
barangay: "Poblacion"    →  barangay: "Poblacion"
municipality: "Calapan"  →  municipality: "Calapan City"
province: "Oriental..."  →  province: "Oriental Mindoro"
                         →  delivery_address: "Purok 1, Barangay Poblacion, Calapan City, Oriental Mindoro" (auto-generated)
```
